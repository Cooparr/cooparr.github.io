TFT Companion

TFT Companion was created to help you improve at Teamfight Tactics, whether that be by providing information about different aspects of the game and its mechanics or helping you to identify the current meta. Below is a list of features found within the app.

  • App Store

Team Compositions

  • Find details on the current best-performing team comps.
  • How to reach each team comp through expected early, mid, and end game champions.
  • View board positions for each champion for the best performance.
  • Customized best items for champions within each team comp.
  • View the exact trait bonuses each team comp provides.
  • Discover detailed champion information by tapping a champion.

Items

  • Browse through the description, attributes & stats of each item.
  • Stay informed on which items are strong via their tier rating.
  • View item recipes to create combined items.
  • Discover or filter out item variants such as Shadow or Radiant items.
  • Quickly see what items you can create by selecting a base item you have.

Champions

  • Quickly view a champion's cost & traits.
  • Discover each champion's unique ability.
  • Easily review the base stats of each champion.
  • Find item suggestions for each champion.
  • Understand which champions are meta via tier rating.
  • Search for champions by name, class, or origin.

Traits

  • Easily browse through each Class & Origin.
  • View each traits description & the benefits they provide.
  • Discover the requirements needed to increase a trait's rank from bronze to gold or platinum if available.

Rolling Rates

  • Discover the total number of champions by tier in the pool.
  • View the percentage chance of seeing a champion by tier in your shop for each player level.

And more...

  • Go back and browse through the previous TFT Sets.
  • Stay up to date with all the latest changes via the patch notes section.
  • Learn different mechanics specific for certain sets, such as 'Galaxies' from Set 3.
  • Personalize your experience via the in-app settings.
